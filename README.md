## üí° The Philosophy Behind This Repository

Welcome to my **Spring Boot Concept Labs**. 

When learning a framework as vast as Spring Boot, it is easy to get lost in "tutorial hell"‚Äîbuilding generic To-Do apps without understanding the internal magic. I decided to take a different approach.

**This repository is not just a collection of apps; it is a documentation of my experiments.** Instead of building one massive, messy application, I am building **concept-wise mini-projects**. Each project in this folder focuses on isolating and mastering a **single specific Spring Boot concept** (such as a specific annotation, a configuration pattern, or an architectural style).

My goal is to move beyond *"It works"* to *"I know exactly **why** it works."*
